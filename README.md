# Empower App

This is a Next.js application created with Firebase Studio. It's a motivational app designed to help users build better habits and stay inspired.

## Core Features

-   **Motivational Messages**: AI-generated motivational quotes.
-   **Book Summaries**: AI-powered summaries of influential books.
-   **Habit Tracking**: Set reminders to break bad habits.
-   **Remembrances**: Morning and evening remembrance prompts (Azkar).
-   **Note Taking**: A simple space for personal notes.
-   **Multi-language Support**: Fully supports English, Arabic, Turkish, Indonesian, and French, including right-to-left (RTL) layout for Arabic.

## Getting Started

To run the project locally:

1.  Install dependencies: `npm install`
2.  Run the development server: `npm run dev`

The app will be available at `http://localhost:9002`.
