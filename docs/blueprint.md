# **App Name**: Empower

## Core Features:

- Motivational Message Generation: Generate and display a random motivational message or saying at the top of the screen, curated by AI for relevance.
- Habit Story: Display a story related to habit formation.
- Book Summary Story: Display a summary of a book related to personal development.
- Morning Remembrance Story: Display a story with morning remembrance prompts.
- Evening Remembrance Story: Display a story with evening remembrance prompts.
- Note Taking: Allow the user to jot down notes.

## Style Guidelines:

- Primary color: Warm amber (#FFB300) to evoke motivation and energy.
- Background color: Pale beige (#F5F5DC) for a calming and clean background.
- Accent color: Soft coral (#FF7F50) to highlight interactive elements.
- Headline font: 'Belleza' sans-serif font for headlines; elegant and fashionable feel. Body text: 'Alegreya' serif font for readability in story content.
- Minimalist design with a focus on readability, ensuring the screen is not cluttered. The motivational message appears prominently at the top.
- Subtle animations for transitions between stories and when the motivational message updates.
